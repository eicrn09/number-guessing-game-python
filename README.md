Number Guessing Game (Python)

This is a simple number guessing game I built while learning Python.

How it works
- The program randomly selects a number between 1 and 100
- The user tries to guess the number
- The program gives feedback:
  - "higher" or "lower"
- The user has limited attempts
- Optional hints are available (even/odd)

Features
- Random number generation
- Input handling
- Conditional logic
- Loop-based gameplay
- Hint system

What I learned
- Using random module
- Writing interactive programs
- Managing game logic
- Handling user input

Note
This project was created in 2023 and reflects my early programming experience.
